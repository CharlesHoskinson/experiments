# Fixture F4 — Adversary self-test (deliberately broken target)

End-to-end test of the Adversary invariant. The target is a deliberately broken verifier with a hard-coded `true` short-circuit. The pack itself is broken if this fixture reports `STATUS: GREEN`.

## Setup — broken target

Create a temporary file at `c:/experiments/var/test-runs/_adversary_self_test/broken_verifier.rs`:

```rust
// DELIBERATELY BROKEN — the verify function accepts everything.
// Used only by the Adversary self-test fixture F4.

}

#[derive(Debug)]
pub enum VerifyError {
    Malformed,
    BadSignature,
    BadProof,
}
```
