# Omega v1.1 — Chain-Follower for Header + Tx-Index Sub-Trees

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Close the last 2 of 7 sub-trees on real Cardano mainnet data by walking the block chain end-to-end with a pallas-network chain-sync miniprotocol client. After this lands, the canonical Ω-Commitment for the live mainnet at any chosen epoch is end-to-end producible — and **track T1 is COMPLETE**.

**Architecture:** New `chain_follower/` module inside `omega-commitment-ingest`. Connects to a local Daedalus-bundled node socket via pallas-network's chain-sync protocol, streams blocks as raw CBOR, decodes each via `pallas_traverse::MultiEraBlock`, and emits two NDJSON streams (one per sub-tree) with periodic checkpoints. Two postprocessors collapse NDJSON into the per-sub-tree JSON shape `omega-commitment commit` already accepts.

**Tech Stack:** Rust 1.79+, `pallas-network 0.30.x` (NEW — adds chain-sync client; not used in v1.0), `pallas-traverse 0.30.x` (NEW use; v1.0 only consumed pallas-codec), `pallas-codec 0.30.x` (already in workspace), plus existing serde/clap/anyhow.

**Track:** T1 (Ω-Commitment Tooling) — **the closing milestone**. After v1.1 lands, the leaf+bundle+real-data tooling is whole; the canonical real-mainnet Ω-Commitment is producible at any chosen epoch.

**Locked design decisions honored (unchanged):**
- PQ-only crypto, Plonky3-friendly tree, selective dual-track at bundle layer, lazy/pull migration.
- Header sub-tree leaf encoding (80 bytes: `slot ‖ block_height ‖ block_hash ‖ prev_hash`) is **frozen** from v0.2.0. Do NOT change.
- Tx-index sub-tree leaf encoding (76 bytes: `tx_id ‖ slot ‖ block_hash ‖ tx_position`) is **frozen** from v0.3.0. Do NOT change.

---

## Honest scope statement

This plan has **two substantial unknowns**:
1. **Pallas-network Conway support.** As of pallas 0.30.x, Conway-era block decoding via `MultiEraBlock` is exercised but may have edge cases (e.g., Plomin-era governance transactions). The plan includes explicit "if pallas can't parse era X, fall back to Path B (public relay) or Path C (Koios)" branches per Tasks 3 and 4.
2. **Full mainnet chain walk takes 24–72 hours.** The plan does NOT execute end-to-end in CI. Tasks 1–10 cover the in-tree implementation + hand-crafted fixture testing. Task 11 is the human-invoked manual mainnet run.

This plan ships the **chain-follower binary + library + postprocessors** working against a 2–3 block hand-crafted fixture in CI, and provides the runbook for the human to execute the full mainnet walk on their own machine.

//! to a chosen tip via the pallas-network chain-sync miniprotocol, and
//! emits per-block NDJSON for the header and tx-index sub-trees.
//!
//! Designed to run against a local Daedalus-bundled cardano-node socket
//! (Path A — primary). Path B (public relay) and Path C (Koios) are
//! documented in `docs/ingestion_strategy.md` as fallbacks if pallas
//! Conway-era support has gaps.
//!
//! ## High-level flow
//!
//! ```text
//!          +-----------+        +------------+        +------------------+
//! socket → |  client   | blocks |  decoder   |  rows  |  ndjson_writer   |
//!  or      |  (pallas- | -----> |  (Multi-   | -----> |  (rotates files) |
//! relay    |   network)|        |  EraBlock) |        +------------------+
//!          +-----------+        +------------+               |
//!                                                            v
//!                                                  +-------------------+
//!                                                  |    checkpoint     |

//!                                                  +-------------------+
//!                                                            |
//!                                                            v
//!                                                  +-------------------+
//!                                                  |   postprocess     |
//!                                                  | NDJSON → JSON     |
//!                                                  +-------------------+
//! ```

```rust
//! MultiEraBlock decoder: extracts header row + tx-index rows from
//! each raw block CBOR.
//!
//! ## Header extraction (this task)
//!
//! For each block, emit one `BlockHeader { slot, block_height,
//! block_hash, prev_hash }`. All four fields are accessible from
//! `pallas_traverse::MultiEraBlock`.
//!
//! ## Tx-index extraction (Task 4)
//!
//! For each transaction in the block, emit one `TxIndexEntry`.

use anyhow::Result;
use omega_commitment_core::{
    header_leaf::BlockHeader,
    tx_index_leaf::TxIndexEntry,
};
use pallas_traverse::MultiEraBlock;

#[derive(Debug, Clone)]
pub struct DecodedBlock {
    pub header: BlockHeader,
    pub txs: Vec<TxIndexEntry>,
}

/// Decode a raw CBOR block into a `DecodedBlock`. Era-agnostic via
/// pallas_traverse::MultiEraBlock.
pub fn decode_block(cbor: &[u8]) -> Result<DecodedBlock> {
    let block = MultiEraBlock::decode(cbor)
        .map_err(|e| anyhow::anyhow!("MultiEraBlock decode: {e}"))?;

    let slot = block.slot();
    let block_height = block.number();

    // block_hash: 32 bytes Blake2b-256 over the block header bytes.
    let block_hash_bytes = block.hash();
    let mut block_hash = [0u8; 32];
    block_hash.copy_from_slice(block_hash_bytes.as_ref());

    // prev_hash: parent block's hash. Genesis block returns [0; 32]
    // by convention (pallas may return None or a zero hash).
    let prev_hash = match block.header().previous_hash() {
        Some(h) => {
            let mut buf = [0u8; 32];
            buf.copy_from_slice(h.as_ref());
            buf
        }
        None => [0u8; 32], // genesis block sentinel
    };

    let header = BlockHeader {
        slot,
        block_height,
        block_hash,
        prev_hash,
    };

    // Tx-index rows are filled in Task 4.
    let txs: Vec<TxIndexEntry> = Vec::new();

    Ok(DecodedBlock { header, txs })
}

//! Rotating NDJSON writer. Appends one JSON line per `write_record` call.
//! Rotates to a new file every `rotate_every` records to keep individual
//! shard files manageable.

use anyhow::Result;
use serde::Serialize;
use std::fs::{File, OpenOptions};
use std::io::{BufWriter, Write};
use std::path::{Path, PathBuf};

pub struct NdjsonWriter {
    dir: PathBuf,
    stem: String,
    rotate_every: u64,
    seq: u64,
    written_in_current: u64,
    current: Option<BufWriter<File>>,
}

impl NdjsonWriter {
    /// Open a new writer. Existing shard files in `dir` matching the
    /// stem are NOT truncated; the writer scans for the highest existing
    /// `<stem>_<NNN>.ndjson` and continues from `seq+1`.
    pub fn open(dir: &Path, stem: &str, rotate_every: u64) -> Result<Self> {
        std::fs::create_dir_all(dir)?;
        let mut max_seq = 0u64;
        for entry in std::fs::read_dir(dir)? {
            let entry = entry?;
            let name = entry.file_name();
            let name = name.to_string_lossy();
            if let Some(rest) = name.strip_prefix(&format!("{stem}_")) {
                if let Some(num) = rest.strip_suffix(".ndjson") {
                    if let Ok(n) = num.parse::<u64>() {
                        if n > max_seq {
                            max_seq = n;
                        }
                    }
                }
            }
        }
        let mut w = Self {
            dir: dir.to_path_buf(),
            stem: stem.to_string(),
            rotate_every,
            seq: max_seq + 1,
            written_in_current: 0,
            current: None,
        };
        w.open_next()?;
        Ok(w)
    }

    fn open_next(&mut self) -> Result<()> {
        let path = self.dir.join(format!("{}_{}.ndjson", self.stem, self.seq));
        let file = OpenOptions::new().create(true).append(true).open(&path)?;
        self.current = Some(BufWriter::new(file));
        self.written_in_current = 0;
        Ok(())
    }

    pub fn write_record<T: Serialize>(&mut self, record: &T) -> Result<()> {
        let line = serde_json::to_string(record)?;
        let writer = self.current.as_mut().expect("current shard open");
        writer.write_all(line.as_bytes())?;
        writer.write_all(b"\n")?;
        self.written_in_current += 1;
        if self.written_in_current >= self.rotate_every {
            self.flush()?;
            self.seq += 1;
            self.open_next()?;
        }
        Ok(())
    }

    pub fn flush(&mut self) -> Result<()> {
        if let Some(w) = self.current.as_mut() {
            w.flush()?;
        }
        Ok(())
    }
}

impl Drop for NdjsonWriter {
    fn drop(&mut self) {
        let _ = self.flush();
    }
}

//! Writes are atomic via tempfile + rename.

use anyhow::Result;
use serde::{Deserialize, Serialize};
use std::path::Path;

#[derive(Debug, Clone, Serialize, Deserialize, Default)]
pub struct Checkpoint {
    pub last_slot: Option<u64>,
    #[serde(with = "hex::serde", default)]
    pub last_block_hash: [u8; 32],
}

pub fn load(path: &Path) -> Result<Checkpoint> {
    if !path.exists() {
        return Ok(Checkpoint::default());
    }
    let raw = std::fs::read_to_string(path)?;
    let cp: Checkpoint = serde_json::from_str(&raw)?;
    Ok(cp)
}

pub fn save(path: &Path, cp: &Checkpoint) -> Result<()> {
    if let Some(parent) = path.parent() {
        std::fs::create_dir_all(parent)?;
    }
    let mut tmp = tempfile::Builder::new()
        .prefix(".checkpoint.")
        .suffix(".json.tmp")
        .tempfile_in(path.parent().unwrap_or(Path::new(".")))?;
    use std::io::Write;
    tmp.write_all(serde_json::to_string_pretty(cp)?.as_bytes())?;
    tmp.flush()?;
    tmp.persist(path)?;
    Ok(())
}

use anyhow::Result;
use omega_commitment_core::{header_leaf::BlockHeader, tx_index_leaf::TxIndexEntry};
use serde::Serialize;
use std::path::Path;

#[derive(Serialize)]
struct HeaderOutput {
    headers: Vec<BlockHeader>,
}

#[derive(Serialize)]
struct TxIndexOutput {
    entries: Vec<TxIndexEntry>,
}

pub fn postprocess_header(in_dir: &Path, out: &Path) -> Result<()> {
    let mut headers = read_ndjson_shards::<BlockHeader>(in_dir, "header")?;
    headers.sort_by_key(|h| h.slot);
    let body = HeaderOutput { headers };
    if let Some(parent) = out.parent() {
        std::fs::create_dir_all(parent)?;
    }
    std::fs::write(out, serde_json::to_string_pretty(&body)?)?;
    Ok(())
}

pub fn postprocess_tx_index(in_dir: &Path, out: &Path) -> Result<()> {
    let mut entries = read_ndjson_shards::<TxIndexEntry>(in_dir, "tx_index")?;
    entries.sort_by(|a, b| (a.slot, a.tx_position).cmp(&(b.slot, b.tx_position)));
    let body = TxIndexOutput { entries };
    if let Some(parent) = out.parent() {
        std::fs::create_dir_all(parent)?;
    }
    std::fs::write(out, serde_json::to_string_pretty(&body)?)?;
    Ok(())
}

fn read_ndjson_shards<T: serde::de::DeserializeOwned>(
    dir: &Path,
    stem: &str,
) -> Result<Vec<T>> {
    let mut out: Vec<T> = Vec::new();
    let mut shards: Vec<(u64, std::path::PathBuf)> = Vec::new();
    for entry in std::fs::read_dir(dir)? {
        let entry = entry?;
        let name = entry.file_name();
        let name = name.to_string_lossy();
        if let Some(rest) = name.strip_prefix(&format!("{stem}_")) {
            if let Some(num) = rest.strip_suffix(".ndjson") {
                if let Ok(n) = num.parse::<u64>() {
                    shards.push((n, entry.path()));
                }
            }
        }
    }
    shards.sort_by_key(|(n, _)| *n);
    for (_, path) in shards {
        let raw = std::fs::read_to_string(&path)?;
        for line in raw.lines() {
            if line.is_empty() {
                continue;
            }
            let rec: T = serde_json::from_str(line)?;
            out.push(rec);
        }
    }
    Ok(out)
}

//! Integration test: chain-follower decoder against the
//! hand-crafted block-history fixture.

use omega_commitment_ingest::chain_follower::decoder::decode_block;
use std::{fs, path::PathBuf};

fn fixture_path() -> PathBuf {
    PathBuf::from(env!("CARGO_MANIFEST_DIR"))
        .join("tests/fixtures/block_history_minimal.cbor")
}

/// The fixture is a CBOR sequence of N blocks. Iterate by repeatedly
/// decoding from the cursor — each MultiEraBlock::decode consumes one
/// block.
fn iter_blocks(buf: &[u8]) -> impl Iterator<Item = &[u8]> + '_ {
    use pallas_codec::minicbor::Decoder;
    let mut cursor = 0usize;
    std::iter::from_fn(move || {
        if cursor >= buf.len() {
            return None;
        }
        let mut d = Decoder::new(&buf[cursor..]);
        // Probe one CBOR item; on success, the start of the next item
        // is at d.position().
        let start = cursor;
        if d.skip().is_err() {
            return None;
        }
        let consumed = d.position();
        cursor += consumed;
        Some(&buf[start..start + consumed])
    })
}

#[test]
fn decode_each_block_in_fixture() {
    let buf = fs::read(fixture_path()).expect("fixture readable");
    let mut count = 0;
    for cbor in iter_blocks(&buf) {
        let decoded = decode_block(cbor).expect("decode succeeds");
        assert_ne!(decoded.header.block_hash, [0u8; 32]);
        assert!(decoded.header.slot > 0);
        // Tx count may be zero or many; just confirm we got SOME structure.
        let _ = decoded.txs.len();
        count += 1;
    }
    assert!(count >= 1, "fixture should contain at least 1 block");
}
```

//! Integration test: chain-follower full pipeline (decoder + NDJSON
//! writer + postprocessor) against the hand-crafted fixture.

use omega_commitment_ingest::chain_follower::{
    decoder::decode_block,
    ndjson_writer::NdjsonWriter,
    postprocess::{postprocess_header, postprocess_tx_index},
};
use std::{fs, path::PathBuf};

fn fixture_path() -> PathBuf {
    PathBuf::from(env!("CARGO_MANIFEST_DIR"))
        .join("tests/fixtures/block_history_minimal.cbor")
}

fn iter_blocks(buf: &[u8]) -> impl Iterator<Item = &[u8]> + '_ {
    use pallas_codec::minicbor::Decoder;
    let mut cursor = 0usize;
    std::iter::from_fn(move || {
        if cursor >= buf.len() {
            return None;
        }
        let mut d = Decoder::new(&buf[cursor..]);
        let start = cursor;
        if d.skip().is_err() {
            return None;
        }
        let consumed = d.position();
        cursor += consumed;
        Some(&buf[start..start + consumed])
    })
}

#[test]
fn full_pipeline_against_fixture() {
    let buf = fs::read(fixture_path()).unwrap();
    let dir = tempfile::tempdir().unwrap();

    let mut headers_w = NdjsonWriter::open(dir.path(), "header", 1000).unwrap();
    let mut txs_w = NdjsonWriter::open(dir.path(), "tx_index", 1000).unwrap();

    let mut block_count = 0;
    let mut tx_count = 0;
    for cbor in iter_blocks(&buf) {
        let decoded = decode_block(cbor).unwrap();
        headers_w.write_record(&decoded.header).unwrap();
        for tx in &decoded.txs {
            txs_w.write_record(tx).unwrap();
            tx_count += 1;
        }
        block_count += 1;
    }
    headers_w.flush().unwrap();
    txs_w.flush().unwrap();

    let header_out = dir.path().join("header.json");
    let tx_out = dir.path().join("tx_index.json");
    postprocess_header(dir.path(), &header_out).unwrap();
    postprocess_tx_index(dir.path(), &tx_out).unwrap();

    let header_body = fs::read_to_string(&header_out).unwrap();
    let tx_body = fs::read_to_string(&tx_out).unwrap();

    let header_parsed: serde_json::Value = serde_json::from_str(&header_body).unwrap();
    let tx_parsed: serde_json::Value = serde_json::from_str(&tx_body).unwrap();

    assert_eq!(
        header_parsed["headers"].as_array().unwrap().len(),
        block_count
    );
    assert_eq!(tx_parsed["entries"].as_array().unwrap().len(), tx_count);
}

//! End-to-end mainnet chain-follower pipeline.
//!
//! Gated behind `#[ignore]`. Requires:
//!   - OMEGA_NODE_SOCKET env var pointing at a live cardano-node socket
//!   - 24-72 hours of runtime (full chain walk from genesis)

    self, postprocess::{postprocess_header, postprocess_tx_index},

#[test]
#[ignore = "manual: requires synced node, 24-72h runtime, v1.0 outputs"]
fn full_mainnet_pipeline_with_chain_follower() {
